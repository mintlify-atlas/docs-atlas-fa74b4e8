Ruby on Rails is a web application framework that includes everything needed to create database-backed web applications according to the Model-View-Controller (MVC) pattern. Rails is designed to make programming web applications easier by making assumptions about what every developer needs to get started.

## The Rails Way

Rails is built on two guiding principles that shape everything you build:

- **Convention Over Configuration** — Rails has opinions about the best way to do many things and defaults to those conventions, so you spend less time on boilerplate and more time on your application.
- **Don't Repeat Yourself (DRY)** — Every piece of knowledge lives in a single, unambiguous place. Less duplication means more maintainable, extensible code.

## Quick install

<Steps>
  <Step title="Install Ruby">
    Rails requires Ruby 3.2 or later. Install it via [rbenv](https://github.com/rbenv/rbenv) or [RVM](https://rvm.io).

    ```bash
    ruby --version
    # => ruby 3.2.0 or later
    ```
  </Step>
  <Step title="Install Rails">
    ```bash
    gem install rails
    ```
  </Step>
  <Step title="Create a new application">
    ```bash
    rails new myapp
    cd myapp
    ```
  </Step>
  <Step title="Start the server">
    ```bash
    bin/rails server
    ```

    Visit `http://localhost:3000` to see your Rails app running.
  </Step>
</Steps>
